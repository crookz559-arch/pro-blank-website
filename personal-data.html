Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с требованиями Федерального закона от 27.07.2006 № 152-ФЗ «О персональных данных» и определяет порядок обработки персональных данных, осуществляемой ООО «ПРО-БЛАНК» (далее — «Оператор»).

2. Правовое основание обработки

Обработка персональных данных осуществляется на основании:

  • Федерального закона № 152-ФЗ «О персональных данных»
  • Согласия субъекта персональных данных
  • Договора, стороной которого является субъект персональных данных
  • Законных интересов Оператора, не противоречащих правам субъекта

3. Состав обрабатываемых данных

Категория данных Состав Цель
Идентификационные Имя, фамилия Идентификация субъекта, обращение
Контактные Телефон, email Связь, уведомления, договор

4. Способы получения данных

  • Заполнение форм на Сайте pro-blank.ru
  • Общение в мессенджерах (WhatsApp, Telegram)
  • Переписка по электронной почте
  • Телефонные переговоры
  • Автоматический сбор технических данных при посещении Сайта

5. Цели обработки персональных данных

  1. Рассмотрение заявок и предоставление консультаций
  2. Подготовка и исполнение договоров на производство
  3. Расчёт стоимости и подготовка коммерческих предложений
  4. Доставка готовой продукции Заказчику
  5. Направление информации о новых услугах (при наличии согласия)
  6. Улучшение качества обслуживания
  7. Соблюдение требований бухгалтерского и налогового законодательства

6. Принципы обработки

  • Законность и справедливость — обработка только на законных основаниях
  • Ограниченность цели — данные используются только для заявленных целей
  • Минимизация — собираем только необходимые данные
  • Точность — данные актуальны и при необходимости обновляются
  • Ограниченное хранение — данные хранятся не дольше, чем необходимо
  • Целостность и конфиденциальность — применяются меры защиты

7. Сроки хранения данных

  • Данные потенциальных клиентов (заявки без договора) — 1 год
  • Данные по исполненным договорам — 5 лет (в соответствии с налоговым законодательством)
  • Технические данные (cookies, логи) — до 1 года
  • Данные для маркетинговых рассылок — до отзыва согласия

8. Передача данных третьим лицам

Оператор может передавать данные:

  • Транспортным и курьерским компаниям для доставки заказов
  • Государственным органам по законным требованиям
  • Платёжным системам при проведении расчётов

Передача данных в иных случаях осуществляется только с письменного согласия субъекта.

9. Меры по защите данных

  • Шифрование передаваемых данных (SSL/TLS)
  • Контроль доступа — только уполномоченные сотрудники
  • Регулярное резервное копирование
  • Инструктаж сотрудников по работе с персональными данными
  • Хранение данных на серверах, расположенных на территории РФ

10. Права субъекта персональных данных

Вы вправе:

  • Запросить сведения об обрабатываемых данных и основаниях обработки
  • Потребовать уточнения или исправления неверных данных
  • Отозвать согласие на обработку данных
  • Потребовать прекращения обработки или удаления данных
  • Обжаловать действия Оператора в Роскомнадзоре (rkn.gov.ru)

11. Порядок реализации прав

Для реализации своих прав направьте письменный запрос по электронной почте info@pro-blank.ru с указанием ФИО, контактных данных и сути обращения. Оператор рассмотрит запрос в течение 10 рабочих дней.

12. Изменения Политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия размещается на Сайте. При существенных изменениях Оператор уведомит субъектов доступными способами.
